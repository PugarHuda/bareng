The invisible stack

Particle · Universal Accountscore
One balance, every chain.
Top up from any token on any chain; spend from a single unified balance. Chain abstraction is the spine — a shared pot is impossible without it.
Magic · Embedded walletcore
Google in, wallet ready.
Social login mints a seedless EOA that owns the account and signs. Web2 onboarding, no MetaMask, no phrase to lose.
ZeroDev · Kernel7702reference
Caps, headed on-chain.
Each member’s cap is an owner-signed 7702 grant, verified cryptographically and enforced app-side today. lib/zerodev.ts is a working Kernel7702 toCallPolicy (USDC.transfer ≤ cap) — the path to chain-enforced caps. (Standalone reference — a different account from the UA; it doesn’t enforce on the UA itself.)
Arbitrum · Settlementcore
The backend you never see.
Every spend settles on Arbitrum One. The user never picks a chain or holds gas — it’s just where the money lands.
Openfort · x402 agent paymentsthe twist
The same capped key is a safe agent wallet.
Point a member’s 7702-capped key at an x402 flow and it pays per-request for a service — an API, compute, a group subscription — bounded by a spend cap so it can’t drain the pot. An autonomous agent with real guardrails. (Reference demo — cap guard tested, settlement abstracted.)

Every round each member pays in; one collects the whole pot — with a verifiable fair draw (keccak-seeded from a public seed) so who collects first can’t be rigged and anyone can recompute it. We didn’t build a shared wallet; we put gotong royong itself on-chain.
